# Excel Sales Dashboard & Analysis

## 📌 Project Overview
This project is an **Excel-based Sales Analysis Dashboard** designed to track and evaluate the performance of sales employees in a company. The workbook includes employee details, their regions of operation, daily sales over a five-day period, and comparisons against targets, with percentage achievement calculated automatically.  

It is implemented using **Microsoft Excel** with **VBA macros** for automation, making sales data analysis and reporting fast and interactive.

---

## 🔑 Features
- 📊 **Sales Tracking:** Records employee IDs, names, and assigned regions.
- 📅 **Daily Sales Records:** Logs sales for five consecutive days.
- 🎯 **Target vs. Achievement:** Calculates total sales, targets, and performance percentage.
- ⚙️ **VBA Automation:** Automates calculations and refreshes reports to save time.
- 📈 **Dynamic Dashboard:** Provides an at-a-glance view of team and individual sales performance.
- 🔍 **Easy Navigation:** Includes slicers, filters, and interactive components for quick analysis.

---

## 🛠️ Technologies Used
- **Microsoft Excel** (Macros Enabled Workbook `.xlsm`)
- **VBA (Visual Basic for Applications)** for automation
- Excel formulas, pivot tables, and charts
